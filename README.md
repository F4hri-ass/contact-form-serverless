# Serverless Contact Form

Aplikasi form kontak berbasis cloud menggunakan arsitektur serverless.

## Fitur
- Website statis sebagai antarmuka pengguna
- Serverless Function untuk memproses data
- Penyimpanan data di cloud database
- Pengiriman email notifikasi otomatis

## Teknologi
- Vercel (Serverless & Hosting)
- Supabase (Cloud Database)
- Resend (Email API)

## Alur Kerja
User mengisi form → data dikirim ke serverless function → disimpan ke database → email notifikasi dikirim.
